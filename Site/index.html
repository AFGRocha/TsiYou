Sobre Nós

Funcionalidades

Este website tem como principal funcionalidade divulgar e promover atividades relativas à Licenciatura de Tecnologias e Sistemas de Informação para a Web. De modo a conseguir promover essas mesmas eventos criamos uma aplicação com o intuito de ajudar todos os discentes desta licenciatura.

Para quem?

Este website distina-se, principalmente, a todos os discentes da Licenciatura de Tecnologias e Sistemas de Informação para a Web, mas também para todos os amantes das tecnologias que estejam interessados nas novas alterações do mundo informático.
